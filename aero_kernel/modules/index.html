Module aero_kernel::modules[−][src]

Expand description

A kernel module is an object file that contains code that can extend the kernel functionality at runtime. When a kernel module is no longer needed, it can be unloaded. Most of the device drivers are used in the form of kernel modules.

Example

fn hello_init() {}
fn hello_exit() {}

aero_kernel::module_init!(hello_init);
aero_kernel::module_exit!(hello_exit);

Structs

ModuleInit

Inner helper structure holding the module init function as a void pointer. This struct is required as we cannot directly store a pointer in the static as it needs to implement Sync.

Functions

init

This function is responsible for initializing all of the kernel modules. Since currently we cannot read the ext2 root filesystem, we link all of the kernel modules into the kernel itself (this is temporary and modules will be loaded from the filesystem in the future).

make_module_init

Inner helper function to make sure the function provided to the module_init macro has a valid function signature. This function returns the passed module init function as a const void pointer.
